# Guitar FingerBoard Quiz

Vue 3 + Vite로 만든 기타 지판 학습 앱입니다.

## 기능

- 표준 튜닝 `E A D G B E`
- `1프렛 ~ 12프렛` 범위 표시
- `# / b`를 제외한 `A B C D E F G`만 출제
- 현재 위치를 지판에서 강조 표시
- 4지선다 객관식 퀴즈
- 정답 수, 정답률, 연속 정답 기록

## 실행 방법

```bash
npm install
npm run dev
```

브라우저에서 안내되는 로컬 주소를 열면 됩니다.

## 빌드

```bash
npm run build
```
